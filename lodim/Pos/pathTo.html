Iterable<Pos> pathTo(
  1. Pos other, {
  2. Path using = lineBresenham,
  3. bool exclusive = false,
})

Returns an iterable of positions that draws a path from this to other.

The default line algorithm is lineBresenham, which is a fast and efficient algorithm for drawing lines between two points in a 2D space. You can also provide your own Path.

Example

final a = Pos(0, 0);
final b = Pos(2, 2);
print(a.pathTo(b)); // => [Pos(0, 0), Pos(1, 1), Pos(2, 2)]

Implementation

Iterable<Pos> pathTo(
  Pos other, {
  Path using = lineBresenham,
  bool exclusive = false,
}) {
  return using(this, other, exclusive: exclusive);
}
