copyRect<E> function

void copyRect<E>(
  1. Rect source,
  2. E get(
    1. Pos
    ),
  3. void set(
    1. Pos,
    2. E
    ), {
  4. Pos target = Pos.zero,
})

Copies each element within source, exclusive of the bottom-right edge, to another grid-like structure.

The get function is called with each position within the bounds, and the return value is passed to the set function with the same position.

It is the caller's responsibility to ensure that the bounds are valid or the behavior is undefined. This function is intended as a low-level utility for grid-like structures; consider asserting or clamping the bounds if necessary before calling this function.

The changes are applied in row-major order.

Example

For the below examples, we use a simple 2-dimensional list as the grid-like structure.

Consider package:sector for a complete implementation.

Copy a grid to another grid

final grid = [
  [1, 2, 3],
  [4, 5, 6],
];

final other = [
  [0, 0, 0],
  [0, 0, 0],
];

copyRect(
  Rect.fromWH(3, 2),
  (pos) => grid[pos.y][pos.x],
  (pos, value) => other[pos.y][pos.x] = value,
);

print(other); // => [[1, 2, 3], [4, 5, 6]]

Copy a sub-region to another grid

final grid = [
  [1, 2, 3],
  [4, 5, 6],
];

final other = [
  [0, 0, 0],
  [0, 0, 0],
];

copyRect(
  Rect.fromLTWH(1, 0, 2, 1),
  (pos) => grid[pos.y][pos.x],
  (pos, value) => other[pos.y][pos.x] = value,
  target: Pos(1, 0),
);

print(other); // => [[0, 1, 2], [0, 0, 0]]

Implementation

void copyRect<E>(
  Rect source,
  E Function(Pos) get,
  void Function(Pos, E) set, {
  Pos target = Pos.zero,
}) {
  // Set each element in the target grid.
  // Assuming the following:
  // 0 0 0 0
  // 0 0 1 2
  // 0 0 3 4
  //
  // Copying into:
  // 0 0
  // 0 0
  //
  // (2, 2) -> (0, 0) if target is (0, 0)
  final srcRect = source;

  // Translate and re-anchor the source rectangle to the target position.
  final dstRect = Rect.fromTLBR(target, target + srcRect.size);

  // Iterate over each position in the source rectangle.
  final srcIt = srcRect.positions.iterator;
  final dstIt = dstRect.positions.iterator;
  while (srcIt.moveNext() && dstIt.moveNext()) {
    set(dstIt.current, get(srcIt.current));
  }
}
