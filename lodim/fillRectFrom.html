fillRectFrom<E> function

void fillRectFrom<E>(
  1. Rect bounds,
  2. void set(
    1. Pos,
    2. E
    ),
  3. Iterable<E> values
)

Fills each element within bounds, exclusive of the bottom-right edge, from an iterable of values.

The set function is called with each position within the bounds, and the corresponding value from values is passed as the second argument (which is iterated in order).

It is the caller's responsibility to ensure that the bounds are valid or the behavior is undefined. This function is intended as a low-level utility for grid-like structures; consider asserting or clamping the bounds if necessary before calling this function.

The changes are applied in row-major order.

Example

For the below examples, we use a simple 2-dimensional list as the grid-like structure.

Consider package:sector for a complete implementation.

Fill a grid with values

final grid = [
  [0, 0, 0],
  [0, 0, 0],
];

fillRectFrom(
  Rect.fromWH(3, 2),
  (pos, value) => grid[pos.y][pos.x] = value,
  [1, 2, 3, 4, 5, 6],
);

print(grid); // => [[1, 2, 3], [4, 5, 6]]

Fill a sub-region with values

final grid = [
  [0, 0, 0],
  [0, 0, 0],
];

fillRectFrom(
  Rect.fromLTWH(1, 0, 2, 1),
  (pos, value) => grid[pos.y][pos.x] = value,
  [1, 2],
);

print(grid); // => [[0, 1, 2], [0, 0, 0]]

Implementation

void fillRectFrom<E>(
  Rect bounds,
  void Function(Pos, E) set,
  Iterable<E> values,
) {
  final pIt = bounds.positions.iterator;
  final vIt = values.iterator;
  while (pIt.moveNext() && vIt.moveNext()) {
    set(pIt.current, vIt.current);
  }
}
