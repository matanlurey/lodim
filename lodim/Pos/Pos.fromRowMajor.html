Creates a position representing index in a grid of width columns.

The position is calculated as if the grid was row-major, i.e. the x-axis is the primary sort key, and the y-axis is the secondary, or the algorithm index = y * width + x.

Note that, like other methods in Pos, there are no assertions that this coordinate is valid, or that width is positive, and the result is undefined if the conditions are not met.

The inverse of this operation is toRowMajor.

Example

final a = Pos.fromRowMajor(5, width: 3);
print(a); // => Pos(2, 1)

Implementation

factory Pos.fromRowMajor(int index, {required int width}) {
  final x = index % width;
  final y = index ~/ width;
  return Pos(x, y);
}
